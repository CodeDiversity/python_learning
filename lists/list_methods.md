## List Methods

### Append: Add item to end of list

first_list =  [1,2,3,4]
first_list.append(5)

Cannot Comma Separate Values.
prints [1,2,3,4,5]

### Extend: Add to the end of a list all values

first_list.extend([5,6,7,8])

would add the each individually.

### Insert
first_list.insert(index, item_to_add)

>>> data = [1,2,3,4]
>>> data.insert (0,2)
>>> data
[2, 1, 2, 3, 4]
>>>
